Useful and easy way to customize your WordPress admin area and the login page.

== Description ==

The plugin has its own admin options page (settings page) where you can:

- add user-friendly horizontal menu and  dashboard metaboxes
- completely remove blogging plarform widgets
- change logo and link of the login page
- customize admin bar menu
- add custom content to the admin footer
- hide notices (core update notice, plugin update notice and theme update notice)
- hide screen options and help tag
- change basic site settings e.g. hide site generator and feed links


It gives your clients a better experience of their new website.

Support for translation!

- Polish translation
- Dutch translation by Jos Wolbers

Plugin is  ready for WordPress 3.6


== Installation ==

1. Upload `cms-admin-area` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go te the admin area options page (Settings-> Admin Area Options)
